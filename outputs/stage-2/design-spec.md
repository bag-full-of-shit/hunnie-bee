# Hunnie-Bee 디자인 명세서

## 1. 디자인 원칙

### 1.1 핵심 가치
1. **심플함**: 최소한의 UI로 최대의 기능
2. **긍정성**: 성취에 집중하는 따뜻한 피드백
3. **직관성**: 한눈에 파악되는 진행 상황
4. **재미**: 벌집을 채워가는 게이미피케이션 요소

### 1.2 디자인 키워드
- 🐝 꿀벌, 벌집
- 🍯 꿀색, 따뜻함
- ⬡ 육각형, 기하학
- ✨ 성취감, 보람

## 5. 컴포넌트 명세

### 5.1 벌집 그리드 (Honeycomb Grid)

**구조**:
- 10열 x 10행 = 100개의 육각형 셀
- 홀수 행은 반 셀 오른쪽으로 오프셋

**셀 크기**:
- 너비: (화면너비 - 32px) / 11
- 높이: 셀너비 × 1.1547 (육각형 비율)

**셀 상태**:
| 상태 | 배경색 | 테두리 | 효과 |
|------|--------|--------|------|
| Empty | Gray-100 | Gray-200 (1px) | - |
| Filled | Honey-400 | Honey-500 (1px) | 그림자 |
| Latest | Honey-300 | Honey-500 (2px) | Glow 효과 |

**애니메이션**:
- 새로 채워질 때: scale 0 → 1.1 → 1 (300ms, spring)
- 탭 시: scale 1 → 0.95 → 1 (150ms)

### 5.2 진행률 표시 (Progress Display)

**레이아웃**:
```
┌─────────────────────────────┐
│      🐝                     │
│     42/100                  │
│    42% 달성                 │
│  58번 더 하면 목표 달성!     │
└─────────────────────────────┘
```

**스타일**:
- 메인 숫자: Display, Honey-700
- 퍼센트: H3, Gray-600
- 응원 메시지: Body, Gray-500

### 5.3 목표 카드 (Goal Card)

**구조**:
```
┌─────────────────────────────┐
│ 📚 책 읽기                 → │
│ ▓▓▓▓▓▓▓░░░ 42/100          │
│ 58일 남음                    │

```

**스타일**:
- 배경: White (라이트) / Gray-800 (다크)
- 모서리: 16px border-radius
- 그림자: 0 2px 8px rgba(0,0,0,0.1)
- 패딩: 16px

**프로그레스 바**:
- 높이: 8px
- 배경: Gray-200
- 채움: Honey-400
- 모서리: 4px

### 5.4 실천 버튼 (Record Button)

**기본 상태**:
- 크기: 가로 100%, 높이 56px
- 배경: Honey-500
- 텍스트: White, Body-L, Bold
- 모서리: 12px
- 그림자: 0 4px 12px rgba(245, 158, 11, 0.3)

**텍스트**: "오늘 했어요! 🐝"

**누른 상태**:
- 배경: Honey-600
- scale: 0.98
- 그림자 감소

**완료 피드백**:
1. Haptic: success
2. 버튼 → 체크 아이콘 변환 (300ms)
3. Confetti 효과 (선택적)
4. 1초 후 원래 상태로

## 8. 모션 & 애니메이션

### 8.1 기본 원칙
- 모든 애니메이션은 60fps 유지
- 과도한 애니메이션 지양
- 목적이 있는 움직임만 사용

### 8.2 타이밍 함수

| 이름 | 값 | 용도 |
|------|-----|------|
| ease-out | cubic-bezier(0, 0, 0.2, 1) | 진입 |
| ease-in | cubic-bezier(0.4, 0, 1, 1) | 퇴장 |
| spring | damping: 15, stiffness: 150 | 바운스 효과 |

### 8.3 주요 애니메이션

| 요소 | 액션 | 애니메이션 | 시간 |
|------|------|------------|------|
| 셀 채우기 | 새 기록 | scale 0→1.1→1 + opacity | 300ms |
| 버튼 | 탭 | scale 1→0.95→1 | 150ms |
| 카드 | 나타남 | translateY 20→0 + opacity | 200ms |
| 모달 | 열기 | scale 0.9→1 + opacity | 250ms |
| 프로그레스 바 | 업데이트 | width 변화 | 400ms |

### 8.4 햅틱 피드백

| 상황 | 타입 |
|------|------|
| 실천 기록 완료 | notificationSuccess |
| 버튼 탭 | impactLight |
| 오류 발생 | notificationError |
| 목표 달성 (100번) | notificationSuccess × 3 |

## 11. 접근성

| 빈 셀 | "빈 셀, 42개 중 N번째" |
