## 벌 상태 시스템 설계

### 상태 정의
```
행복 (Happy)     : 포만감 ≥ 70% AND 애정도 ≥ 70%
보통 (Normal)    : 포만감 ≥ 40% OR 애정도 ≥ 40%
시무룩 (Sad)     : 포만감 < 40% AND 애정도 < 40%
걱정 (Worried)   : 3일 이상 교감 없음
```

### 수치 시스템
| 수치 | 범위 | 증가 | 감소 |
|------|------|------|------|
| 포만감 (Fullness) | 0-100 | 꿀 주기 +30 | 매일 -10 |
| 애정도 (Affection) | 0-100 | 쓰다듬기 +20 | 매일 -5 |

### 상태별 벌 반응
| 상태 | 표정 | 대사 예시 |
|------|------|-----------|
| 행복 | 😊 반짝반짝 | "오늘도 같이 힘내자!" |
| 보통 | 🙂 평온 | "오늘 하루 어때?" |
| 시무룩 | 😢 축 처짐 | "좀 외로웠어..." |
| 걱정 | 🥺 눈물글썽 | "어디 갔어...? 걱정했어" |

## 데이터 모델

### 벌 상태 (BeeState)
```typescript
interface BeeState {
  name: string;           // 벌 이름
  fullness: number;       // 포만감 (0-100)
  affection: number;      // 애정도 (0-100)
  lastFedAt: Date | null; // 마지막 꿀 준 시간
  lastPetAt: Date | null; // 마지막 쓰다듬은 시간
  lastInteractionAt: Date;// 마지막 교감 시간
  honeyCount: number;     // 보유 꿀 개수
}
```

### 상태 계산 함수
```typescript
function getBeeStatus(state: BeeState): 'happy' | 'normal' | 'sad' | 'worried' {
  const daysSinceInteraction = getDaysDiff(state.lastInteractionAt, new Date());

  if (daysSinceInteraction >= 3) return 'worried';
  if (state.fullness >= 70 && state.affection >= 70) return 'happy';
  if (state.fullness >= 40 || state.affection >= 40) return 'normal';
  return 'sad';
}
```
