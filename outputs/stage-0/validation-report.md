## 결정 사항

### 1. 이미지 에셋
- **MVP**: 이모지로 구현 (🐝😊😢🥺)
- **이후**: 사용자가 직접 이미지 에셋 추가 예정

### 2. 기존 BeeStatus와의 관계
- **선택**: B) 기존 유지 + 새 탭에 별도 시스템
- **홈 탭**: 기존 BeeStatus (목표 진행률 기반) 유지
- **벌 탭**: 새로운 교감 시스템 (포만감/애정도 기반)
- **장점**: 기존 기능 영향 없이 안전하게 추가 가능

## 개발 범위 확정

### MVP (P0) - 1차 개발
1. **벌 탭 UI** - 새 탭 추가
2. **BeeState 데이터 모델** - 포만감, 애정도, 꿀 저장
3. **꿀 획득 로직** - 목표 완료 시 꿀 +1
4. **꿀 주기 액션** - 꿀 소모 → 포만감 증가
5. **상태 계산** - 행복/보통/시무룩/걱정
6. **일일 감소** - 앱 시작 시 수치 감소

### P1 - 2차 개발
1. 쓰다듬기 액션
2. 상태별 이모지/텍스트 변화
3. 벌 이름 설정

### P2 - 향후
1. 이미지 에셋 적용
2. 꾸미기 아이템
3. 벌 성장/진화
