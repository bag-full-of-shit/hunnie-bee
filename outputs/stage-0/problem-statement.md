# Step 0.2: Problem Statement (문제 정의서)

## 현재 문제

### 1. 보상 부재
- 목표를 달성해도 "했다" 버튼 누르는 것 외에 얻는 게 없음
- 성취감이 즉각적으로 느껴지지 않음

### 2. 감정적 연결 없음
- 벌 상태가 바뀌어도 사용자에게 의미가 없음
- "그래서 뭐?" 상태
- 벌과 나의 관계가 형성되지 않음

### 3. 지속 동기 부족
- 앱을 계속 켜야 할 이유가 약함
- 습관 트래킹만으로는 장기적 동기부여가 어려움
- 쉽게 잊혀지는 앱이 됨

---

## 타겟 사용자

**"귀여운 걸 좋아하면서, 습관 형성이 어렵고, 혼자 동기부여가 안 되는 사람"**

### 페르소나
- 습관 앱 여러 개 깔았다 지운 경험 있음
- 귀여운 캐릭터에 마음이 약함
- 혼자서는 의지력이 부족하다고 느낌
- 누군가(무언가)가 나를 기다리면 더 잘함

---

## 핵심 감정 (2가지 모두)

### 1. 귀여움 → 챙기고 싶음
> "벌이 너무 귀여워서 뭔가 해주고 싶어"

- 꿀 주기, 쓰다듬기 등 케어 욕구 자극
- 외형 변화로 보상감 제공

### 2. 미안함 → 열심히 해야겠다
> "벌이 시무룩해서 미안해... 내일은 꼭 해야지"

- 적절한 죄책감으로 행동 유도
- 원망이 아닌 걱정의 느낌

**두 감정의 조합**:
- 귀여움으로 진입 → 미안함으로 유지
- 긍정 + 부정 동기가 균형을 이룸

---

## 이 기능이 해결하는 것

| 문제 | 해결 방법 |
|------|-----------|
| 보상 부재 | 목표 달성 → 꿀 획득 → 벌에게 줄 수 있음 |
| 감정적 연결 없음 | 벌이 나를 걱정하고, 내가 벌을 돌봄 |
| 지속 동기 부족 | 벌 보러 앱 열게 됨, 시무룩하면 마음 아픔 |
